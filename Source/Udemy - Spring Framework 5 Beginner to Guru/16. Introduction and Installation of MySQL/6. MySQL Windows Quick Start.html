MySQL Editions

There are several editions for MySQL. Primarily MySQL Enterprise Edition and MySQL Community Edition.

For the purposes of this course, we will be using the MySQL Community Edition.

The MySQL Enterprise Edition is a paid version of MySQL, which comes with support from Oracle. The functionality for writing SQL for the two editions are the same.

MySQL Windows Installation

MySQL has a Windows Installation program you can download to install MySQL natively on your machine. Most recent versions of Windows are supported. You can find the Windows installer here.

There are two flavours of the installer. A smaller one if you are connected to the internet, and a complete one you can download to install MySQL offline. If your internet performance is slow, try using the offline installer. 

To Install MySQL on Windows:

  1. Download the Windows Installer.
  2. Follow the instructions to install 'MySQL Server'.
  3. Optionally, install MySQL Workbench.
  4. Other MySQL components are optional.

MySQL Editions

There are several editions for MySQL. Primarily, MySQL Enterprise Edition and MySQL Community Edition.

For the purposes of this course, we will be using the MySQL Community Edition.

The MySQL Enterprise Edition is a paid version of MySQL, which comes with support from Oracle. The functionality for writing SQL for the two editions are the same.

MySQL Ubuntu Installation

To install MySQL on Ubuntu:

  1. Run the following commands as root or prefix the commands below with sudo .
  2. Update the package index on your server.

    3. apt-get update 

  3. Install the package.

    4. apt-get install mysql-server-5.7 

Note: If you are not sure about the version, you can omit the version and run sudo apt-get install mysql-server . This will install the latest version for your Linux distribution. 

To install the latest MySQL version not included in the APT package repository:

  1. Install the newer APT package repository from the MySQL APT repository page.

    2. wget http://dev.mysql.com/get/mysql-apt-config_0.6.0-1_all.deb 

  2. Go through the prompt to select and apply the specific MySQL version to install.
  3. Update the package index on your server.

    4. sudo apt-get update 

  4. Install the package.

    5. sudo apt-get install mysql-server 

Once installation is complete, MySQL should run automatically. You can check the status of MySQL using this command.

service mysql status 

MySQL CentOS Installation

To install MySQL on CentOS:

  1. Update your system using: sudo yum update 
    2. ###liMySQL Yum Repository page. 
  2. Select the release package for your platform.
  3. Download the package. (Note: if wget is not found, install with the command - yum install wget 

    4. $ wget https://dev.mysql.com/get/mysql57-community-release-el6-11.noarch.rpm UPdate 

  4. Update system dependencies (again). sudo yum update  
  5. Install the package.

    6. $ sudo rpm -ivh mysql57-community-release-el6-11.noarch.rpm 

  6. Install MySQL.

    7. $ sudo yum install mysql-server 

  7. Start the MySQL daemon.

    8. $ sudo systemctl start mysqld 

  8. Confirm that MySQL is running.

    9. $ sudo systemctl status mysqld

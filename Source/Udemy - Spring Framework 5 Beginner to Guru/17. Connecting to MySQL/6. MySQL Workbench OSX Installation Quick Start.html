MySQL Workbench for OSX is distributed as a DMG file named mysql-workbench-community-version-osx.dmg , where version  is the MySQL Workbench version.

To install MySQL Workbench on OSX:

  1. Access the Download MySQL Workbench page.
  2. Select macOS from the Select Operating System drop-down list.
  3. Download the DMG file.
  4. Double-click the downloaded file. The installation wizard will guide you through the installation process.
  5. Once installation completes, you can launch MySQL Workbench from the Applications folder.
  6. To launch MySQL Workbench from the command line, run this command.

open MySQLWorkbench.app
